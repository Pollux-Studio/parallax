# Parallax

**Trust & Transparency for Ubuntu Touch**

Parallax is a privacy-first, on-device trust intelligence layer that helps users
understand the trustworthiness of installed applications on Ubuntu Touch.

## Philosophy

Parallax does not scan, block, or police applications.
It explains app behavior using signals already present in the Ubuntu Touch
security model.

## Features (v1)

- System-wide trust overview
- Per-app trust score (0–100)
- Human-readable risk explanations
- Fully offline and privacy-respecting
- Ubuntu Touch native (QML / Clickable)

## Tech Stack

- QML / QtQuick
- JavaScript
- Clickable
- Click manifests & metadata

## Privacy

- No internet access
- No analytics
- No background monitoring
- 100% on-device
